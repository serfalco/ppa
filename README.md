# PPA — Pulso Productivo Argentino

Refundación de PPA: publicación económica automática para lectores no especialistas.

El proyecto se encuentra en la **Etapa 0: definición y documentación**. Todavía no comenzó la implementación del sitio nuevo.

## Principios

- Dos ediciones automáticas: Desayuno y Merienda.
- Resúmenes propios con fuente y enlace original.
- Explicaciones económicas en lenguaje claro.
- Datos con fecha, fuente, método y estado de vigencia.
- Costos mínimos y herramientas abiertas.
- Cloudflare como plataforma pública.
