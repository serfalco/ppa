# Modelo editorial

## 1. Flujo automático

```text
Fuente → extracción → normalización → deduplicación → clasificación
       → resumen → controles → selección de edición → publicación
```

El editor interviene por excepción, no como paso obligatorio.

## 2. Unidad básica: noticia resumida

Campos mínimos:

| Campo | Obligatorio | Descripción |
|---|---:|---|
| Título original | Sí | Título de la fuente, limpiado sólo de basura técnica |
| Título PPA | No | Reescritura cuando el original sea confuso o clickbait |
| Resumen PPA | Sí | Síntesis propia |
| Por qué importa | No | Consecuencia o contexto útil |
| Fuente | Sí | Nombre normalizado |
| URL original | Sí | Enlace canónico |
| Publicación original | Sí | Fecha y hora cuando estén disponibles |
| Lectura PPA | Sí | Momento de captura |
| Categoría | Sí | Taxonomía única |
| Estado | Sí | Publicada, oculta, corregida o descartada |
| Edición | Sí | Desayuno, Merienda o ambas |

## 3. Resúmenes automáticos

Objetivo recomendado: entre 70 y 130 palabras. Deben permitir comprender el hecho sin reemplazar la pieza original.

Reglas:

- Usar palabras propias.
- Conservar cifras, sujetos y fechas esenciales.
- No añadir antecedentes no presentes salvo que estén identificados como contexto PPA.
- No inferir causalidad.
- No copiar más de una frase breve.
- No usar “según expertos” sin identificar quiénes.
- No afirmar que algo ocurrió si la fuente habla de una posibilidad.
- Finalizar con atribución visible, no dentro del cuerpo del resumen.

## 4. “Por qué importa”

Debe responder una de estas preguntas:

- ¿A quién afecta?
- ¿Qué cambia?
- ¿Qué indicador puede mover?
- ¿Qué antecedente ayuda a entenderlo?
- ¿Qué deberíamos mirar después?

Si la respuesta es obvia o especulativa, se omite.

## 5. Selección de cada edición

La edición prioriza:

1. Relevancia económica argentina.
2. Novedad real.
3. Calidad y cercanía de la fuente.
4. Diversidad de categorías.
5. Evitar repetición del mismo hecho.
6. Impacto productivo, institucional o social.

No se garantiza una cuota fija por categoría.

## 6. Jerarquía de fuentes

Para hechos y cifras:

1. Organismo o documento primario.
2. Institución sectorial responsable.
3. Medio especializado.
4. Medio general.
5. Cuenta personal identificada.

Una cuenta de X puede alertar sobre un dato, pero el sistema debe buscar el documento original cuando exista.

## 7. Correcciones

Toda corrección material debe registrar:

- Contenido anterior.
- Contenido nuevo.
- Motivo.
- Fecha.
- Responsable.

Las correcciones pequeñas de ortografía no necesitan aviso público; las correcciones de hechos o cifras sí.

## 8. Columnas

Son opinión firmada. Deben diferenciarse visualmente de La Data y de Datos PPA.

Campos:

- Título.
- Bajada.
- Autor.
- Fecha.
- Cuerpo.
- Estado: borrador o publicado.
- Imagen opcional.

Primera columna prevista: **“Romper el chancho, soltar el canuto”**. Texto pendiente.

## 9. TXT-Stream

Tipos posibles:

- Transcripción editada.
- Resumen de entrevista.
- Adaptación de video o audio.
- Texto generado desde apuntes propios.

Debe conservarse el origen y explicarse el grado de edición.

## 10. EconoTuits

Es una selección, no una reproducción completa de X.

Grupos:

- Oficiales: organismos y funcionarios en ejercicio.
- Institucionales: universidades, cámaras, consultoras y centros de estudio.
- Personales: economistas, analistas y periodistas.

Primera versión: 12 a 15 cuentas totales. Si el acceso gratuito deja de funcionar, la sección puede pasar a curación manual o suspenderse sin afectar el sitio.

## 11. Definiciones fáciles

Cada definición reutilizable tendrá:

- Nombre.
- Explicación de una línea.
- Explicación ampliada.
- Método.
- Fuente metodológica.
- Conceptos relacionados.
- Fecha de revisión.

Formato visual sugerido:

> **En criollo:** explicación corta, concreta y sin jerga innecesaria.

## 12. Avisos editoriales

PPA debe indicar:

- Que utiliza automatización e inteligencia artificial.
- Que enlaza y resume contenidos de terceros.
- Que los errores pueden ser informados y corregidos.
- Que los datos son informativos y no constituyen asesoramiento financiero.
