# ADR-001 — Refundar en lugar de migrar literalmente

Estado: aceptada  
Fecha: 2026-07-01

## Contexto

El proyecto actual reúne aprendizaje valioso, pero mezcla generación, datos, edición, publicación y archivos heredados. Migrarlo sin cambios trasladaría sus problemas a otro proveedor.

## Decisión

Construir un producto nuevo y conservar el proyecto actual como referencia. Migrar únicamente contenidos, fuentes y criterios aprobados.

## Consecuencias

- Se reduce deuda heredada.
- Deben revalidarse fuentes y datos.
- El lanzamiento será gradual.
- No se conservará el archivo histórico completo.
